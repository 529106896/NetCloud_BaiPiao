# NetCloud_BaiPiao
自学Vue期间突发奇想写的一个白嫖网易云vip歌曲的前端项目

## 2022年2月8日04:26:37

纯前端，采用 HTML5 + Vue2 + axios 编写，懒得写css，所以很丑



主要借助网易云公开的api（以及部分自己搜索得到的api）实现，无需登录验证



目前的功能有：

- 搜歌（最多列出30个，暂时不能翻页）
- 查看歌曲封面
- 听歌（可听需vip的歌曲）
